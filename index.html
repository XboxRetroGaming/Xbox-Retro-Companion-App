Frequently Asked Questions

Why the app was called W10M Group?

This app created as support for the WP community and by time extended to be a very advanced tool.

Who is the developer of this app?

Bashar Astifan is the developer of WUT, but we cannot forget the others support and help (read About page).

What Windows builds are supported?

Any build equal or above 14393 (Desktop) including W10M, Works perfect as well in Windows 11

What is Commander?

Commander is a very advanced scripts engine, works like any automate app download WUT and read the documentation for more details

Is Telnet required for Commander?

Yes and no, if you have non-telnet scripts will work fine.. but for telnet scripts you need to have telnet connection on your pc, check settings page->telnet for basic telnet server

Why the built-in browser important?

The browser made to help any developer with many advanced tools in one place, like you can post data.. inspect elements and there is a built-in security module to block ads and any unsecure request.. and more..

What is BeDrive cloud?

BeDrive is PHP (cloud server management) allows you to build you own cloud storage using your own hosting or Dropbox, Amazon and more..
